# 📊 Development Tracking System

> A full-stack web application to manage tasks, track progress, and improve team productivity.

---

## 📌 Overview
The Development Tracking System helps admins assign tasks and monitor progress, while users can update their work status in real time.

---

## 🚀 Features

### 🔐 Authentication
- Admin Login
- User Login & Signup

### 🧑‍💼 Admin Dashboard
- Manage users  
- Assign tasks  
- Track progress (Pending, In Progress, Completed)  
- View task reports  

### 👩‍💻 User Dashboard
- View assigned tasks  
- Update task status  
- Filter tasks  
- Export reports  

---

## 🛠️ Tech Stack

Frontend:

Backend:
